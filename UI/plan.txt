3. Custom CSS Variables
Define CSS variables for colors, font sizes, and spacings to make your CSS reusable and easy to maintain.
css
Copy code

  --primary-color: #3498db;
  --secondary-color: #2ecc71;
  --padding: 16px;
}

  background-color: var(--primary-color);
  padding: var(--padding);
}
